# Descrição da atividade:

Este repositório contém o conteúdo do laboratório 8, da disciplina Introdução a Ciência da Computação.

O objetivo da atividade foi utilizar uma LLM (Large Language Model) para
realizar pequenas tarefas, exercitando a escrita de prompts e utilizando essas ferramentas de forma consciente.

# Conteúdo do repositório:

relatorio.pdf - versão final do relatório, contendo:

    Escolha da ferramenta utilizada;
    Explicação conceitual (prompt simples vs. prompt elaborado);
    Geração de código, testes e análise crítica;
    Melhoria do prompt e comparação dos resultados;
    Verificação crítica de uma das respostas;
    Seção "Uso responsável de LLMs";
    Seção "Reflexão sobre o uso de IA Generativa";
    Registro detalhado de uso da LLM (prompts, respostas e verificações).

prompts.txt - todos os prompts utilizados durante a atividade.
